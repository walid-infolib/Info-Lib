Withholding Tax

Withholding tax on invoices

To Configure:


Nothing to configure, just install the module.

Withholding tax is a model that enable the user to create specific taxes. Those taxes are specified in the payment functionality. Under invoice, accounting module we can find the submenu of withholding tax under the configuration menu.

By default there are 5 withholding taxes 1%, 5%, 10% and 15%.

We can manage those withholding taxes (add, delete, modify) as needed.

We can access payment functionality under Customers or Vendors menus. So we can specify the withholding tax, the withholding journal used to make the payment and finally manage the amount, the withholding amount also the amount net. Also we need to set the withholding journal since it's a mondatory field.

To respect the proces we need to set tunisia as the country of the current company .

At the other hand we can make multiple customers or vendors invoices payment and specify if a group payment or pay each one at its own.

If we didn't specify to make a group payment after the registration of the payment we will have multiple payments each one is linked to an invoice and journal entries of withholding tax.

Finally by accessing a payment in which we have already applied a withholding tax we can print a report that summarize what has happened.
