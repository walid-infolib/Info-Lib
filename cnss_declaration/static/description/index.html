Our module allows the preparation and processing of the quarterly salary declaration of the CNSS (National Social Security Fund) in accordance with the latest specifications in force in Tunisia.

To Configure:

This module requires the installation of hr_payroll module
Nothing to configure, just install the module.

Company Configuration

Add The Social Security Number and the other necessary information.

Employees Configuration

Add The Social Security Number of each employee and their professional category.


Add the necessary informations to the type of contract such as the CNSS salary rate, Employer cnss rate, The Professional Training Tax,...

CNSS Declaration

Now just access to payroll application and click on CNSS Declaration in the Payslips menu.


This is the tree view, we have the quarter of the declaration , the year, date of declaration and their status " Draft or Done".


Compute the quarter of the year and contract type to report a simplified declaration.
We can print this declaration report in pdf format.
The status of the declaration is Draft by default.
If we click on the Generate File button, the status will be 'Done' and the file will be uneditable, so we cannot change any information in this case.
Also a text file will be added in the attachment part containing the recording trace of the magnetic support of teledeclaration salary.


You find the generated file in the attachment part of the chatter.

The total of all monthly salaries for each table.
For the last page we add the general total of all tables.
